# 🔬 Собирающая линза — Лабораторная работа по оптике

Интерактивный веб-сайт для выполнения лабораторной работы по определению оптической силы и фокусного расстояния собирающей линзы.

> **Демо:** [optics-lab.onrender.com](https://optics-lab.onrender.com)

---

## Описание

Учебная платформа, которая позволяет студентам выполнять лабораторную работу, проходить тест и решать задачи — а преподавателю просматривать результаты в реальном времени.

### Возможности для студента

- **Лабораторная** — ввод расстояния *d*, автоматический расчёт *f*, *F*, *D*, *Г* с пошаговым решением и интерактивным чертежом хода лучей (Canvas)
- **Тест** — 10 вопросов по теме с проверкой и подсветкой правильных/неправильных ответов, возможность пройти повторно
- **Задачи** — 5 расчётных задач с проверкой ответа, подсказками после 4 неудачных попыток и системой баллов

### Возможности для преподавателя

- Просмотр результатов всех учеников (лабораторная, тест, задачи)
- История всех попыток по каждому разделу
- Отображение способа решения задач (самостоятельно / с подсказкой)
- Общий балл каждого ученика из 20
- Смена пароля через личный кабинет

---

## Система оценивания

| Раздел | Баллы |
|--------|-------|
| Тест (10 вопросов) | 1 балл за вопрос = **10 баллов** |
| Задачи (5 задач) | 2 балла без подсказки / 1 балл с подсказкой = до **10 баллов** |
| **Итого** | **20 баллов** |

## Технологии

- **Backend:** Python, Flask
- **Frontend:** HTML5, CSS3, JavaScript, Canvas API
- **База данных:** PostgreSQL (Neon)
- **Деплой:** Render (Free Tier)

---

## Запуск локально

```bash
# Клонировать репозиторий
git clone https://github.com/lizaTimofeeva/optics-lab.git
cd optics-lab

# Установить зависимости
pip install -r requirements.txt

# Указать строку подключения к БД
export DATABASE_URL="postgresql://..."

# Запустить
python app.py
```

Сервер запустится на `http://localhost:5000`
